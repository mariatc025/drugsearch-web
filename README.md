# DrugSearch website

DrugSearch is a web platform that allows users to explore detailed information about over 1,100 drugs, including side effects, interactions, subcategories, and manufacturers.

## Usage

1. Open the [DrugSearch Website](https://formacio.bq.ub.edu/~u254549/drugsearch-web/index.html)
2. Login using the demo accounts or create an account:

**User:** `user@gmail.com` / `user1`

**Admin:** `admin@gmail.com` / `admin1 `  

4. Explore the platform features:
   - Users can search and save drugs, and update their profiles.  
   - Admins can access the admin panel and manage users.
